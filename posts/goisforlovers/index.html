MySQL 数据库

#MySQL 数据库 什么是数据库？

 数据库(Database)是按照数据结构来组织，存储和管理数据的，建立在计算机存储设备上的仓库。

mysql 常用数据库引擎：innodb muisam

常用的数据库有哪些？

   mysql(百万级的数据库)，被Oracle收购

   access(微软，淘汰)

   sql  server(微软新版)

    Oracle 甲骨文(比mysql更高级的数据库)
